## ⚠️ PROBLEMAS COMUNS

### "Unable to connect to Metro"
**Soluções:**
1. Verifique se está na mesma WiFi
2. Desative VPN (se estiver usando)
3. Tente reiniciar o Expo Go
4. Verifique firewall do Windows

### "Network request failed"
**Soluções:**
1. Backend pode estar bloqueado
2. Tente acessar no navegador: `http://192.168.1.6:5175`
3. Se não abrir, firewall está bloqueando

### App não carrega
**Soluções:**
1. Aguarde 1-2 minutos (primeira vez demora)
2. Feche e abra o Expo Go
3. Tente inserir a URL novamente
